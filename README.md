# CNC Router / Plasma Quoting Tool

A Streamlit app for configuring CNC machine sizes, calculating costs, and generating quotes based on area-based pricing.

## Features

- Configure machine sizes with predefined configurations
- Calculate extrusion and steel frame costs
- Area-based profit calculation
- Optional plasma unit selection
- Compare all configurations at once
- Generate detailed parts lists and quotes

## Deployment

This app is deployed on Streamlit Community Cloud.
